Construct é um editor de jogos 2D baseado em HTML5, desenvolvido pela Scirra Ltda. É destinado primeiramente para não-programadores, permitindo a criação rápida de jogos, por meio do estilo drag-and-drop usando um editor visual e um sistema de lógica baseada em comportamento.

Juntamente com o meu grupo QWERTY, desenvolvemos um jogo 2D em plataforma

SOBRE O JOGO: o Jogo "SANTA`ANA ADVENTURE" conta a historia de um aluno que mata a aula de filosofia e busca juntar moeda/dinheiro para ir no Bar Curtir com seus amigos, porem seus professores estaram como obstaculos pois buscam levar o aluno de volta para a sala

Nosso jogo "SANTA`ANA ADVENTURE" possui 3 fases com cenarios diversificados. O cenario conta com um efeito parallax tambem

O personagem possui 3 vidas ao longo do jogo e ao morrer quando o personagem perde o jogo, aparece uma mensagem na tela "GAME OVER".

O inimigo em nosso seria o professor tentandoo impedir do aluno prosseguir pelas fases.

O jogo contem tambem com um sistema de som. Colocamos audio no jogo quando o personagem morre, leva dano e quando consegue obter 1 moeda. O jogo contem com um sistema de armazenamento(somatorio) de moedas ao longo das fases.

Processing 3 é uma linguagem de programação de código aberto e ambiente de desenvolvimento integrado, construído para as artes eletrônicas e comunidades de projetos visuais com o objetivo de ensinar noções básicas de programação de computador em um contexto visual e para servir como base para cadernos eletrônicos


Utilizei a ferramenta do Processing para criar um jogo da cobrinha.

O meu jogo possui uma biblioteca de audio Minim.
A cada ponto que é coletado pela cobra, é adicionado mais 1 ponto para o seu tamanho.
Sua pontuacao é armazenada ao decorrer do jogo e ao morrer seus pontos sao resetados.

O jogo é de facil jogabilidade.

Microsoft Visual Studio é um ambiente de desenvolvimento integrado da Microsoft para desenvolvimento de software especialmente dedicado ao .NET Framework e às linguagens Visual Basic, C, C++, C# e F#.
Juntamente com Carlos Eduardo e Matheus, fizemos um jogo de batalha naval.

Primeiramente posicionamos os nossos navios no tabuleiro. Apos ajustarmos 3 navios dos 16 posicoes liberadas, partimos para o ataque.

No canto superior esquerdo tem opcoes para qual coordenadas devemos atingir, por exemplo: A1, B2, C3 e assim sucessivamente. Temos 16 posicoes aonde podemos atingir o oponente.
Apos cada acerto que demos no oponente, conta um ponto para o jogador.

Vence quem acertar mais navios dentre 5 rounds.
